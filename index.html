My stack:

HTML

HTML is the standard markup language for creating web pages and web applications. With HTML, you can create a website that is accessible to all users, regardless of their device or browser.

CSS

CSS is a style sheet language used for describing the look and formatting of a document written in HTML. CSS allows you to separate the content (HTML) from the presentation (CSS).

Java script

JavaScript is a programming language that enables you to create dynamically updating content, control multimedia, animate images, and much more. JavaScript is a high-level, interpreted programming language that conforms to the ECMAScript specification.

PHP

PHP is a server-side scripting language that is widely used to create dynamic web pages. PHP is a free and open-source scripting language. PHP is designed to be easy to learn and easy to use.

SQL

SQL is a relational database management system (RDBMS) that uses SQL queries to retrieve and manipulate data stored in a database. SQL is a structured query language (SQL) used to communicate with a database.

bootstrap

Bootstrap is a free and open-source front-end framework for designing websites and web applications. Bootstrap includes HTML and CSS-based design templates for typography, forms, buttons, tables, navigation, modals, image carousels, and many other components.

figma

Figma is a user-friendly design tool that enables anyone to create and share interactive designs. Figma is a collaborative design platform that allows teams to work on designs together, making it easier to create high-quality, user-friendly products.

photoshop

Photoshop is a raster graphics editor developed by Adobe Systems, which is used for designing images, editing images, and creating digital images. Photoshop is a powerful tool that allows you to manipulate images, create animations, and create vector graphics.

blender

Blender is a free and open-source 3D computer graphics software for creating, editing, and rendering 3D graphics. Blender is a powerful tool that allows you to create complex 3D models, animations, and scenes.

Photographing

Photography is the process of capturing images using a camera or other photographic equipment. Photography is a creative and artistic field that allows people to capture moments, memories, and experiences.
